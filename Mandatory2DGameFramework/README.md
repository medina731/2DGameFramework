# Mandatory2DGameFramework

A lightweight 2D game framework created for educational purposes.  
The framework provides core components for building creatures, items, worlds,  
and combat logic in a structured and extendable way.

# Features
- Creature system with inheritance (Elf, Fairy, etc.)
- Attack and Defense item system using Decorator and Composite patterns
- CreatureFight class for handling combat logic
- World system for placing and interacting with creatures
- XML configuration support
- Clean architecture with SOLID principles
